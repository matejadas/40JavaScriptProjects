40 Vanilla JavaScript Projects

1 How to create a Color Flipper

La idea es crear una página con un color de fondo aleatorio con mensaje que muestre el color actual y un botón que sirva para cambiar el color de fondo a uno aleatorio.

2 How to create a counter

Se mostrará un contador a 0 por defecto y debajo tendrá tres botones: incrementar, disminuir y poner a 0. Los números positivos se mostrarán en verde y los negativos en rojo. Mientras se mantenga el botón pulsado el contador subirá o bajará.

3 How to create a Review carousel

Se creará un carrusel de reseñas. Habrá unas cuantas ya hechas y se navegará entre ellas con flechas. También habrá un botón para ir a una de ellas al azar.
